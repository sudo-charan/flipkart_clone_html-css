# Flipkart Clone

![Flipkart Clone](https://sudo-charan.github.io/flipkart_clone_html-css/)

## Description

This project is a clone of the popular e-commerce platform, Flipkart. It aims to replicate the core functionalities and design elements of Flipkart using HTML and CSS.


## Technologies Used

- **Frontend:** HTML, CSS

## Installation

1. Clone the repository:
    git clone https://github.com/sudo-charan/flipkart_clone_html-css.git

2. Navigate to the project directory:
    cd flipkart_clone_html-css

3. Open `index.html` in your preferred web browser.
